
I am keeping this directory just in case
I ever get another Mac with OSX. These scripts
are the most complex examples.

### OSX Example Scripts

	* GNocmp.sh		GNU-Ada on OSX [no Xcode g++] (default)
	* GNocmp_so.sh	GNU-Ada on OSX using shared liboal.so [uses Xcode g++]
	* AComp.sh		AdaCore Ada on OSX [no Xcode-g++]

	* ocmp_ac.sh	AdaCore Ada on OSX [no Xcode-g++]
	* ocmpx.sh		AdaCore Ada on OSX [uses Xcode-g++]
